Events#

Mint#

event Mint(address indexed to, uint value);

Emitted each time a wrapped bitcoin is minted.

  • to is the receiver of the minted tokens.
  • value is the amount of the minted tokens.

Burn#

event Burn(address indexed to, uint value);

Emitted each time a wrapped bitcoin is burnt.

  • to is the owner of the burnt tokens.
  • value is the amount of the burn tokens.

State-Changing Functions#

ccTransfer#

function ccTransfer(    bytes4 version,    bytes memory vin,    bytes calldata vout,    bytes4 locktime,    uint256 blockNumber,    bytes calldata intermediateNodes,    uint index,    bool payWithHGT) external returns(bool);

Executes cross-chain transfer requests.

  • version is the Bitcoin transaction version number.
  • vin is the Bitcoin transaction inputs.
  • vout is the Bitcoin transaction outputs.
  • locktime is a time or block number.
  • blockNumber is number of block that includes user's request.
  • intermediateNodes is the merkle inclusion proof of user's request.
  • index is the position of user's request in the block.
  • payWithHGT is true if user wants to pay relay fee using HGT.

mintAfterFinalization#

function mintAfterFinalization(bytes32 txId) external returns(bool);

Executes a previously submitted cross-chain transfer request after it gets finalized.

  • txId is the transaction id of the submitted request.

instantCCTransferWithPermit#

function instantCCTransferWithPermit(    address signer,    bytes memory signature,    address receiver,    uint instantTokenAmount,    uint deadline) external returns(bool);

Executes instant cross-chain transfer requests.

  • signer is the address of the user who wants to perform an instant transfer.
  • signature is the signature of the signer under the request data.
  • receiver is address of instant tokens receiver.
  • instantTokenAmount is the amount of tokens that signer wants to transfer.
  • deadline is the last block number that the request can be executed in it.
