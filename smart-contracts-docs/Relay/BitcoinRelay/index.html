addHeaders#

function addHeaders(bytes calldata _anchor, bytes calldata _headers) external returns (bool);

Adds new block headers to storage after validating.

  • _headers is the chain of new block headers.
  • _anchor is the block header immediately preceeding the new block headers

checkTxProof#

function checkTxProof(  bytes32 txid,  uint blockHeight,  bytes calldata intermediateNodes,  uint index,  bool payWithHGT,  uint neededConfirmations) external returns (bool);

Checks inclusion of txid in the block header of height blockHeight.

  • blockHeight is height of block that includes txid.
  • intermediateNodes is the merkle inclusion proof of transaction.
  • index is the position of transaction in the block.
  • payWithHGT is true if user wants to pay relay fee using HGT.
  • neededConfirmations is the required number of confirmations of txid.
