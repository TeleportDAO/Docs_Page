FastPool

Code#

FastPool.sol

Events#

AddLiquidity#

event AddLiquidity(address user, uint wrappedBitcoinAmount);

Emitted each time liquidity is added to fast pool.

RemoveLiquidity#

event RemoveLiquidity(address user, uint wrappedBitcoinAmount);

Emitted each time liquidity is removed from fast pool.

Read-Only Functions#

owner#

function owner() external view returns (address);

Returns the address that is allowed to set contract parameters.

wrappedBitcoin#

function wrappedBitcoin() external view returns (address);

Returns address of wrapped bitcoin contract.

fastRouter#

function fastRouter() external view returns (address);

Returns address of fast router contract.

fastFee#

function fastFee() external view returns (uint);

Returns fee percentage for borrowing from the fast pool.

fastLimit#

function fastLimit() external view returns (uint);

Returns maximum amount that can be borrowed from the fast pool per block.

neededConfirmations#

function neededConfirmations() external view returns(uint);

Returns required number of confirmations for a fast request.

totalRequestedAmount#

function totalRequestedAmount(uint blockNumber) external view returns(uint);

Returns borrowed amount of wrapped bitcoin in block blockNumber.

totalWrappedBitcoin#

function totalWrappedBitcoin() external view returns(uint);

Returns wrapped bitcoin balance of fast pool.

State-Changing Functions#

addLiquidity#

function addLiquidity(address user, uint wrappedBitcoinAmount) external returns(uint);

Adds wrappedBitcoinAmount of wrapped bitcoin to fast pool and mints fast pool token for user.

removeLiquidity#

function removeLiquidity(address user, uint fastPoolTokenAmount) external returns (uint);

Burns fastPoolTokenAmount of fast pool token and sends proportional amount of wrapped bitcoin to 'user'.

fastTransfer#

function fastTransfer(address user, uint amount, uint blockNumber) external returns(bool);

Transfers amount of wrapped bitcoin to user in block blockNumber.

  • This function can only be called by fast router.
